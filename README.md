#  Credit Scoring Model

## 📌 Project Overview

This project predicts whether a person is creditworthy using Machine Learning techniques.

## 🎯 Objective

To classify customers as **Low Risk (Good Credit)** or **High Risk (Bad Credit)** based on financial data.

##  Algorithms Used

* Logistic Regression
* Decision Tree
* Random Forest (Best Model)

## 📊 Dataset Features

* Age
* Income
* Home Ownership
* Employment Length
* Loan Intent
* Loan Grade
* Loan Amount
* Interest Rate
* Credit History

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC

## 🎨 UI

An interactive dashboard built using **Gradio** with custom styling and background.

## 🚀 How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the notebook or UI:

```
python app.py
```

## 🏁 Result

Random Forest achieved the best performance and is used for prediction.

## 📌 Conclusion

This model helps financial institutions assess credit risk effectively.
